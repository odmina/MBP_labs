Twój pierwszy projekt badawczy

8.5 Rekodowanie

Czasem może zdarzyć się, że będziecie chcieli zamienić wartości jakiejś zmiennej. Poniżej procedura, która pozwala zmienić wartości zmiennej przyklad “Zdecydowanie nie”, “Raczej nie”, “Raczej tak”, “Zdecydowanie tak” na 1, 2, 3, 4.

moje_dane <- moje_dane %>%
  mutate(przyklad = as.factor(przyklad)) %>%
  mutate(przyklad = fct_recode(przyklad,
    "1" = "Zdecydowanie nie",
    "2" = "Raczej nie",
    "3" = "Raczej tak",
    "4" = "Zdecydowanie tak"))

UWAGA! Kiedy zamieniacie zmienną przyklad na faktor, R ustawia sobie to, w jakiej kolejności występują poziomy i robi to opierając się na kolejności alfabetycznej. Żeby zobaczyć, jaka jest kolejność poziomów, możecie użyć funkcji levels(). Dowiecie się wtedy, że poziomy mają swój porządek, który wygląda tak: Raczej nie, Raczej tak, Zdecydowanie nie, Zdecydowanie tak. Jak zamienicie je na 1, 2, 3, 4, to porządek pozostanie taki sam, czyli: 2, 3, 1, 4 (poziomy będą miały tę samą kolejność, tylko nazwy będą inne). Jeżeli potem zastosujecie funcję as.numeric(), to na liczbę zostanie zamieniona widoczna liczba, tylko ukryty poziom faktora…

Można pokonać ten problem na dwa sposoby. Pierwszy polega na tym, by ustalić poziomy faktora w zadanej kolejności:

moje_dane <- moje_dane %>%
  mutate(przyklad = as.factor(przyklad)) %>%
  mutate(przyklad = fct_recode(przyklad,
    "1" = "Zdecydowanie nie",
    "2" = "Raczej nie",
    "3" = "Raczej tak",
    "4" = "Zdecydowanie tak")) %>%
  mutate(przyklad = factor(przyklad, levels = c("1","2","3","4")))

Inny sposób polega na zastosowaniu innej funkcji, jeżeli macie starszą wersję paczki dplyr:

moje_dane <- moje_dane %>%
  mutate(przyklad = case_when(
    przyklad == "Zdecydowanie nie" ~ 1,
    przyklad == "Raczej nie" ~ 2,
    przyklad == "Raczej tak" ~ 3,
    przyklad == "Zdecydowanie tak" ~ 4))

Jeżeli macie najnowszą lub relatywnie nową wersję paczki dplyr (1.1.2), to możecie skorzystać z jeszcze innej funkcji:

moje_dane <- moje_dane %>%
  mutate(przyklad = case_match(przyklad,
    "Zdecydowanie nie" ~ 1,
    "Raczej nie" ~ 2,
    "Raczej tak" ~ 3,
    "Zdecydowanie tak" ~ 4))
